# 라우팅 프로토콜



## 개요&#x20;

> 라우팅 프로토콜은 라우터끼리 자신들의 정보를 교환하며 패킷이 이동할 최적을 찾기 위한 프로토콜 입니다.&#x20;



## 종류&#x20;

> AS 내부에서 수행되느냐, 외부에서 수행되느냐에 따라 종류를 나눌 수 있습니다.&#x20;

### IGP

> RIP와 OSPF가 있습니다. 이 프로토콜은 최적의 경로를 선정하는 과정에서 거리 벡터가 사용되느냐 링크 상태가 사용되느냐로 구분됩니다.&#x20;
